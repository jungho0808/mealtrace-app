# MealTrace

MealTrace는 현재 위치 주변에서 직접 가서 먹을 식당 하나를 추천하고, 식사 후 만족/보통/불만족 평가를 다음 추천에 반영하는 미니멀 음식 선택 앱입니다.

## 프로젝트 목표

음식을 고르는 데 쓰는 시간을 줄이고, 사용자가 실제로 방문한 식사 경험을 바탕으로 다음 추천을 조금씩 더 잘 맞추는 것을 목표로 합니다.

MealTrace는 많은 선택지를 비교하는 앱이 아니라, 지금 갈 만한 식당 하나를 빠르게 정해 주는 앱입니다.

## v1 범위

- 현재 위치 주변의 직접 방문 식당 추천
- 추천 결과 1개 제공
- 추천 결과에 식당명, 음식 종류, 예상 가는 시간, 예상 가격 표시
- 사용자가 `이걸로 할게요`를 누르면 선택 기록 저장
- 선택 후 외부 지도 앱으로 이동
- 앱 재실행 시 미평가 식사에 대해 만족/보통/불만족/안 먹었어요 평가 요청
- 평가 결과를 다음 추천에 반영

## v1 제외 범위

- 배달 추천
- 포장 추천
- 앱 안 지도 표시
- 리뷰 비교
- 여러 후보를 나열해 비교하는 탐색형 화면
