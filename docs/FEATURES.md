# MealTrace 기능 목록

## v1 핵심 기능

### 1. 현재 위치 기반 추천

사용자의 현재 위치 주변에서 직접 방문 가능한 식당 하나를 추천합니다.

표시 정보:

- 식당명
- 음식 종류
- 예상 가는 시간
- 예상 가격

### 2. 추천 결과 선택

사용자는 추천 결과가 마음에 들면 `이걸로 할게요`를 누릅니다.

처리 내용:

- 선택한 식당 기록 저장
- 해당 식사를 미평가 상태로 저장
- 외부 지도 앱으로 이동

### 3. 외부 지도 앱 이동

MealTrace 안에서 지도를 직접 표시하지 않고, 선택한 식당을 찾아갈 수 있도록 외부 지도 앱으로 이동합니다.

v1에서는 앱 내 지도 탐색을 제공하지 않습니다.

### 4. 미평가 식사 확인

앱 재실행 시 이전에 선택했지만 아직 평가하지 않은 식사가 있으면 먼저 평가를 요청합니다.

평가 선택지:

- 만족
- 보통
- 불만족
- 안 먹었어요

### 5. 평가 기반 추천 반영

식사 후 평가를 다음 추천에 반영합니다.

기본 방향:

- 만족한 음식 종류나 가격대는 추천 우선순위에 긍정적으로 반영
- 불만족한 음식 종류나 식당은 추천 우선순위 하향
- 보통은 약한 신호로 반영
- 안 먹었어요는 만족도 신호가 아닌 미방문 기록으로 처리

## v1에서 하지 않는 기능

- 배달 식당 추천
- 포장 가능 식당 추천
- 앱 안 지도 표시
- 리뷰 비교
- 여러 추천 후보 목록 표시
- 식당 상세 페이지
- 사용자 리뷰 작성
- 친구 공유 또는 소셜 기능
- 예약 기능
- 결제 기능

## 사용자 경험 원칙

- 화면은 적고 결정은 빠르게 만든다.
- 선택지를 늘리기보다 하나의 추천 품질에 집중한다.
- 평가 입력은 한 번의 탭으로 끝낼 수 있게 한다.
- 지도와 길찾기는 사용자가 익숙한 외부 지도 앱에 맡긴다.
